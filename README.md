# File Protection Utility

## Objective

The File Protection Utility is a Python program that converts file content into a protected format and restores it when required.

## Features

* Protects text using Base64 encoding
* Restores protected content using decoding
* Simple and easy-to-use interface
* Demonstrates basic data protection concepts

## Technologies Used

* Python 3
* Base64 Module

## How It Works

1. User chooses to protect or restore content.
2. Text is converted into an encoded format.
3. Encoded content can be stored securely.
4. The program can decode and restore the original content.
